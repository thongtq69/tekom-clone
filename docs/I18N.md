# I18n — Quick Reference

> Chi tiết đầy đủ ở [`AGENTS.md`](../AGENTS.md). File này là cheat-sheet thao tác nhanh.

## Cấu trúc

```
src/dictionaries/
├── vi.json    ← Default. Source of truth cho TypeScript Dictionary type.
├── en.json    ← Cùng key với vi.json, khác value
└── index.ts   ← getDictionary(lang), hasLocale(lang), Locale, locales, defaultLocale
```

## Thêm 1 đoạn text mới

```jsonc
// vi.json
"myFeature": {
  "title": "Tiêu đề mới",
  "desc": "Mô tả..."
}

// en.json — CÙNG key
"myFeature": {
  "title": "New title",
  "desc": "Description..."
}
```

```tsx
// page.tsx
const dict = await getDictionary(lang);
return <h1>{dict.myFeature.title}</h1>;
```

## Thêm 1 page mới

```tsx
// src/app/[lang]/my-route/page.tsx
import { getDictionary, hasLocale } from "@/dictionaries";
import { notFound } from "next/navigation";

export default async function MyPage(props: PageProps<"/[lang]/my-route">) {
  const { lang } = await props.params;
  if (!hasLocale(lang)) notFound();
  const dict = await getDictionary(lang);

  return <main>{dict.myRoute.title}</main>;
}
```

Đăng ký vào nav: thêm vào `navItems` trong `src/components/Header.tsx` + thêm `dict.nav.myRoute` key trong cả 2 file dictionary.

## Thêm ngôn ngữ mới (vd Trung 中文)

1. `src/dictionaries/zh.json` — copy structure từ `vi.json`, dịch values
2. `src/dictionaries/index.ts`:
   ```ts
   const dictionaries = {
     vi: () => import("./vi.json")...,
     en: () => import("./en.json")...,
     zh: () => import("./zh.json").then(m => m.default as Dictionary),
   };
   export const locales: Locale[] = ["vi", "en", "zh"];
   ```
3. `src/proxy.ts`: thêm `"zh"` vào `const locales`
4. `src/components/Header.tsx`: thêm vào dropdown:
   ```tsx
   {(["vi", "en", "zh"] as Locale[]).map(...)}
   ```

## Link nội bộ — LUÔN có prefix lang

```tsx
// ❌ SAI
<Link href="/contact">Contact</Link>

// ✅ ĐÚNG
<Link href={`/${lang}/contact`}>{dict.nav.contact}</Link>
```

Trong client component, lấy `lang` từ props (truyền xuống từ server) hoặc `usePathname().split("/")[1]`.

## Verify khi xong

- Mở `http://localhost:3000/vi` — toàn tiếng Việt
- Mở `http://localhost:3000/en` — toàn tiếng Anh
- Bấm switcher VI/EN — chuyển đúng ngôn ngữ, giữ nguyên path
- Mở `http://localhost:3000/` — auto redirect sang `/vi`
- Console không có error
